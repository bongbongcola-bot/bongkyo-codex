Naver Blog Draft Studio

장소와 사진만 있으면 네이버 블로그 초안을 바로 만드는 웹페이지

촬영장소, 샘플사진, 키워드, 특징을 입력하면 키워드를 포함한 제목과 네이버 블로그 업로드용 본문 초안을 생성합니다. 결과는 html, md, text 형식으로 바로 전환할 수 있습니다.

입력

네이버 SEO를 고려한 제목과 본문 생성을 위한 기본 정보

샘플사진 미리보기

최대 5장까지 업로드할 수 있고, 생성 결과 본문에는 사진이 들어갈 위치가 파일명과 함께 표시됩니다.

출력

제목과 블로그글을 원하는 형식으로 확인하고 복사

생성 제목

아직 생성된 제목이 없습니다.

생성 결과는 네이버 블로그용 정보형 문단으로 정리되며, 본문 안에 샘플사진 배치 위치가 함께 표시됩니다.
